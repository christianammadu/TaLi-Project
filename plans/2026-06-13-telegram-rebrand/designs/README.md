# TaLi rebrand — 3 directions to compare + lock

Three **genuinely different** visual directions for the rebrand (name stays **TaLi**). Each
is a self-contained hi-fi landing mockup with the dual **Continue on WhatsApp** /
**Continue on Telegram** CTA. Open each in a browser (double-click the `.html`).

The WhatsApp green / Telegram blue appear **only** on their CTA buttons in every direction —
the brand colour is TaLi's own, not the channels'.

| | **D-01 · Market Ledger** | **D-02 · Precise Ledger** | **D-03 · Bold Talk** |
|---|---|---|---|
| File | `D-01-market-ledger.html` | `D-02-precise-ledger.html` | `D-03-bold-talk.html` |
| Lineage | Kenya Hara warmth × Nigerian-market craft | Müller-Brockmann / Pentagram Swiss grid | Big-type editorial (Sagmeister-bold) |
| Feel | Warm · human · trustworthy | Precise · serious · data-credible | Confident · memorable · young |
| Canvas | Warm paper `#FBF7EF` | Clean white | Dark warm `#15110D` |
| Accent | Terracotta `#C2562F` | Deep green `#0B6E4F` | Saffron `#F4B23E` |
| Type | Fraunces (serif) + Hanken Grotesk | Space Grotesk + JetBrains Mono | Bricolage Grotesque (oversized) |
| Hero artefact | Chat bubble → ledger lines | Auto-generated statement + sparkline | Oversized "Just talk." wordmark |
| Motion | Calm / static | Static, grid-precise | **Scroll-driven** chat→ledger→numbers |
| Best when | We want approachable + locally credible | We want "serious finance" trust | We want a bold, ownable brand |
| Risk | Could read "soft" to fintech buyers | Could read "cold" to a market trader | Bold dark can tire on long pages |

## How each tells the "chat → ledger → numbers" story
- **D-01** — side-by-side: a real chat on the left resolves into a warm ledger card on the right; 3 calm "how it works" cards below.
- **D-02** — the *statement* is the hero (P&L + sparkline); the 3 steps are a Swiss grid framed as parse → validate → book (leans on the audit-trail strength).
- **D-03** — a scroll-driven 3-act narrative (Act 1 conversation · Act 2 ledger · Act 3 the big number), riffing on the existing `design/landing-wireframes/` A/B/C wireframes.

## Recommendation
**D-01 (Market Ledger)** is the safest fit for the audience (Nigerian SMEs, "credible not Silicon-Valley") while still distinctive. **D-03 (Bold Talk)** is the most ownable/memorable and demos best for a hackathon video. **D-02** is the pick if "trustworthy finance" should lead. A strong hybrid: **D-01's warmth + D-03's wordmark confidence**.
